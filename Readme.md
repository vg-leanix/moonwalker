# Installation

1. Clone repo:
 ```bash 
 git clone https://github.com/vg-leanix/moonwalker
 ```
2. change into the directory:
 ```bash
 cd moonwalker
 ```
3. Run Docker-Compose to spin up app:
 ```bash
 docker-compose up --build
 ```
4. Go to [localhost](http://localhost) to see the app
